# Immune Defense Arcade

Proyecto web estático de un prototipo educativo de inmunología, pensado para navegador y celular.

## Archivos

- `index.html`: estructura principal de la página.
- `styles.css`: estilos visuales y layout responsive.
- `game.js`: lógica del juego en canvas.

## Cómo abrirlo

1. Abre `index.html` en tu navegador.
2. Para una prueba local más estable, puedes usar cualquier servidor estático simple y abrir la carpeta del proyecto.

## Publicarlo en GitHub Pages

1. Sube estos archivos a un repositorio en GitHub.
2. Confirma que `index.html` esté en la raíz del repositorio.
3. En GitHub, ve a `Settings` > `Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama principal y la carpeta `/ (root)`.
6. Guarda los cambios y espera a que GitHub publique el sitio.

## Notas

- No usa frameworks ni build step.
- La estructura actual es compatible con GitHub Pages y otros hosts estáticos como Netlify o Cloudflare Pages.
